# Парсер товаров с сайта dental-first.ru
Вытаскивает названия товаров и их цены, а потом сохраняет всё в файл `result.txt`.

#Что нужно установить

Перед запуском нужно поставить две библиотеки:
requests — чтобы скачивать веб-страницы
beautifulsoup4 — чтобы из HTML доставать данные

#Для установки нужно ввести 
pip install requests beautifulsoup4


#Как это работает?
1. создаются 4 потока, каждый качает свою страницу
2. каждый поток ищет блоки `.set-card.block` — это карточки товаров
3. внутри ищется название (`a.di_b.c_b`) и цена (`.set-card__price`)
4. всё собирается в один общий список
5. итог записывается в файл
