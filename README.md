# Парсер товаров с сайта dental-first.ru
Вытаскивает названия товаров и их цены, а потом сохраняет всё в файл `result.txt`.

#Что нужно установить

Перед запуском нужно поставить две библиотеки:
requests — чтобы скачивать веб-страницы
beautifulsoup4 — чтобы из HTML доставать данные

#Для установки нужно ввести 
pip install requests beautifulsoup4


#Как это работает?
1. создаются 4 потока, каждый качает свою страницу
2. каждый поток ищет блоки `.set-card.block` — это карточки товаров
3. внутри ищется название (`a.di_b.c_b`) и цена (`.set-card__price`)
4. всё собирается в один общий список
5. итог записывается в файл



## Сравнение асинхронного и многопоточного сокет-серверов
Проект содержит два сервера — асинхронный и многопоточный, которые принимают путь к директории и возвращают количество строк в каждом файле.
Также включён клиентский скрипт для тестирования скорости работы.

2.1_async_server.py — асинхронный сервер на asyncio
2.2_multithread_server.py — многопоточный сервер
2.3_client_test.py — клиент + тест производительности

Запуск
1. Запустите асинхронный сервер: python 2.1_async_server.py
2. Запустите многопоточный сервер (в другом окне): python 2.2_multithread_server.py
3. Запустите тестовый клиент: python 2.3_client_test.py

Скрипт создаст тестовые файлы, подключится к обоим серверам и выведет сравнение времени работы.
